ApptInNow.com

Marketing small business appointments and giving Users appointments NOW!!!

Introducing the ultimate web application
for advertising and booking service based appointments

THE SMALL BUSINESS PROBLEM: Small business needs the ability to advertise open appointments for the day that have been left open. It's hard to fill in same day appointments with walk-ins only.

FINDING SAME DAY APPOINTMENTS: It's hard and time-consuming calling multiple businesses asking for their availability, cost, and quality.

Solution: ApptInNow.com gives small business the ability to market same day appointments to the public while letting the public to access those appointments in a convenient and simple web app that is adaptable for mobile and PC users.

Small Business/ Public use

Small Business: All Small business will have to register with a preset template that ask for business name, address, phone, email, general cost, web site, and yelp link.

The Public: No registration needed they can choose service type and then view available appointments from small business

Calendar setup/appointment setup

Small Business: Once registration template is setup, the small business will have the option to choose a custom calendar that meets their needs and then publishing the availability using the chosen calendar

The Public: Once a appointment has been chosen, the customer will have the choice to enter his her her email to be contacted by the business owner for any potential changes and appointment reminder.

Database/Cloud Storage

DynamoDB on AWS(Amazon Web Service) will be used for the exchange of information and keeping all data stored along with having real time data information exchange.

Security and Customer Protection

Amazon web services will also provide the encryption and user authentication to maintain all users information and exchange of information safe.

Tech Stack/Implementation Tools

Eclipse: IDE(integrated development environment),

Maven: is a build automation tool that allows a project to build using its project object model (POM) and a set of plugins that are shared by all projects using Maven, providing a uniform build system.

Apache Tomcat: allows the implementation of Java Servlets and JavaServer Pages (JSP) to promote an effective Java server environment.

Amazon Web Services: A few AWS are being used which are EC2 sever, DynamoDB database, Amazon Cognito User Pool for authentication.

Trello: Project management software that allows me to keep track of current builds and needed timely objectives.

SpringMVC: The Spring Web MVC framework provides Model-View-Controller (MVC) architecture and ready components that can be used to develop flexible and loosely coupled web applications. The MVC pattern results in separating the different aspects of the application (input logic, business logic, and UI logic), while providing a loose coupling between these elements.

Proof of Concept

Planning always goes perfect because there is a expected output to your input but many of ApptInNow.com plans have failed when trying to be implemented. This would have never been known with out a proof of concept. The proof of concept is to develop a simple web page that can be implemented using the Full Tech Stack that was describe. This means using Eclipse to build a dynamic website that mobile users can easliy view using html,css, servlets, using spring as my model view controller, then having Maven package all my plugins and dependencies into one Web Archive file(.war) then setup a AWS EC2 server and initializing the server environment with Java 8 and Tomcat.
